CSS

Que es?

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico que se utiliza para definir la apariencia de los sitios web. Permite controlar el color, la fuente, la posición, las animaciones y los efectos visuales de los elementos de una página web.

JavaScript

Que es?

JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.

Atributos CSS

Que es?

Los atributos en CSS son propiedades que se le asignan a un elemento HTML para proporcionar información adicional. Se pueden usar para aplicar estilos a elementos basados en los atributos que contienen.

nombre descripcion Valores posibles Ejemplo de uso Imagen representativa Fuente de informacion
Font-size Define el tamaño del texto en un elemento.
  • Absolutos: xx-small, x-small, small, medium, large, x-large, xx-large.
  • Relativos: larger, smaller.
  • Medidas específicas:
    • px (píxeles, fijo).
    • em (relativo al tamaño de la fuente del elemento padre).
    • rem (relativo al tamaño de la fuente del html).
    • % (porcentaje relativo).
    • vw (porcentaje del ancho de la ventana).
    		 Cambiar el tamaño de la letra,definir el tamaño, doblar el tamaño base, crear un tamaño relativo dependiendo del html Referencias:MDN - font-size
    Border-Radius Permite redondear las esquinas de un elemento.
  • 0px (sin bordes redondeados).
  • px, % (tamaño en píxeles o porcentaje).
  • border-radius: 50%; (forma circular).
    		•  Cambiar las esquinas ligeramente y redondearlas, asi mismo redondear los bordes de las imagenes Referencias:MDN - border-radius
    Display Controla cómo se muestran los elementos en la página
  • block (Elemento ocupa toda la línea).
  • inline (Elemento ocupa solo el contenido).
  • inline-block (Mezcla de inline y block).
  • none (Oculta el elemento).
    		•  Convertir el span en un bloque Referencias:MDN - display
    Flexbox Flexbox permite organizar elementos dentro de un contenedor de forma flexible.
  • display: flex; (Activa Flexbox en el contenedor).
  • justify-content: (Controla alineación horizontal).
  • align-items: (Controla alineación vertical).
    		•  Al usarlo podemos alinear los elementos dentro de un bloque y asi ubicarlos a nuestro gusto Referencias:CSS Tricks - Flexbox
    Position Controla cómo se posiciona un elemento en la página.
  • static (Valor por defecto, posición normal).
  • relative (Relativo a su posición original).
  • absolute (Relativo al contenedor padre con position: relative;).
  • fixed (Permanente en la pantalla, incluso al hacer scroll).
  • sticky (Se mantiene fijo hasta que se desplaza fuera de su contenedor).
    		•  Ubicar a gusto la ubicacion del elemento, su tamaño, su distancia de ciertas partes de la pag y cuanto espacio ocupa, por ejemplo centrar un elemento o ponerlo muy cerca del lado derecho Referencias:MDN - position
    box-shadow Agrega una sombra a los bordes de un elemento.
  • none (Sin sombra).
  • X Y Desenfoque Expansión Color.
    		•  Definir el tamaño de la sombra de ciertos elemntos y asi mismo un nivel de desenfoque a nuestro gusto Referencias:MDN - box-shadow
    overflow Controla qué pasa cuando un contenido es más grande que su contenedor.
  • visible (El contenido se muestra fuera del contenedor).
  • hidden (Oculta el contenido que se sale del contenedor)
  • scroll (Agrega barras de desplazamiento).
  • auto (Muestra barras de desplazamiento solo cuando es necesario).
    		•  Manejamos el contenido que no entra en su contenedor ya que tiene un tamaño mayor y asi mantenemos un control en la pag web, pudiendo esconderlo o mostrandolo de todas formas Referencias:MDN - overflow
    z-index Controla qué elemento se coloca encima de otro.
  • Un número (z-index: 1; → Se superpone sobre z-index: 0;).
  • auto (Orden normal de la página).
    		•  ubica los elemntos en el plano z, se usa para cuando necesitamos que un elemento este por encima de otro Referencias:MDN - z-index
    opacity Define la opacidad de un elemento.
  • 1 (100% visible).
  • 0.5 (50% transparente).
  • 0 (Totalmente invisible).
    		•  Lo usamos para cuando necesitamos que un elemento sea un poco mas transparente o simplemente invisible Referencias:MDN - opacity
    margin y padding
  • margin: Controla el espacio fuera del elemento.
  • padding: Controla el espacio dentro del elemento.
    		•
  • margin: 10px; (Mismo margen en todos los lados).
  • margin: 10px 20px; (Arriba/abajo 10px, izquierda/derecha 20px).
  • padding: 5px 10px 15px 20px; (Arriba, derecha, abajo, izquierda).
    		•
  • Usamos el margin cuando necesitamos cambiar ciertas distancias de nuestro elemento respecto a lo que esta fuera de este
  • usamos el padding para controlar las distancias dentro de nuestro elemento y asi mantener un control total de nuestra pag y la ubicacion de lo que agregamos
    		•  Referencias:
  • MDN - margin
  • MDN - padding
    		•
    text-align Define la alineación del texto dentro de un elemento.
  • left (Alineado a la izquierda, por defecto).
  • right (Alineado a la derecha).
  • center (Centrado).
  • justify (Justifica el texto para que ocupe todo el ancho disponible).
    		•  La usamos para mover nuestro texto a nuestro gusto, ubicarlo en cualquier parte del renglon Referencias:MDN - text-align
    background-image Permite establecer una imagen como fondo de un elemento.
  • none (Sin imagen de fondo).
  • url('imagen.jpg') (Imagen de fondo).
    		•  Para cuando necesitamos una imagen sin embargo no es el tema principal, entonces se ubica en el fondo para no usarla de manera directa Referencias:MDN - background-image
    cursor Cambia el cursor cuando pasa sobre un elemento.
  • default (Cursor normal).
  • pointer (Cursor de mano, útil para botones).
  • text (Cursor de selección de texto).
  • wait (Cursor de "cargando").
    		•  Lo usamos para cuando necesitamos la atencion del usuario sobre cierto elemento, asi cuando pasa por ese elemento el cursor cambia y llama su atencion. Referencias:MDN - cursor
    visibility Controla si un elemento es visible o no.
  • visible (Se muestra normalmente).
  • hidden (Oculta el elemento, pero sigue ocupando espacio).
    		•  Para cuando necesitamos que cierto elemento no desaparezca pero si deje de ser visible Referencias:MDN - visibility
